# 数据分析指南

## 统计指标解读

### 数值型列（Numeric Columns）
- **均值 vs 中位数（Mean vs Median）**：两者差异大说明数据分布偏斜（常见于收入、价格等字段）
- **最小值/最大值范围（Min/Max range）**：范围过大可能存在异常值（outliers）
- **缺失值（Missing values）**：缺失超过 10% 需要重点调查

### 分类型列（Categorical Columns）
- **唯一值数量（Unique values）**：唯一性过高可能是 ID 字段
- **高频值（Top values）**：分布严重不均可能暗示数据质量问题
- **缺失值（Missing values）**：考虑"缺失"本身是否具有业务含义

## 报告撰写原则
1. 先讲洞察，再讲数字——不要堆砌数据
2. 突出异常——意料之外的发现往往最有价值
3. 尽量做对比——孤立的数字缺乏上下文
4. 给出建议——每个洞察都应指向可能的下一步行动
